Chapter 4: Networking and the Internet
Term Definitions

Network Classifications
  • PAN - A personal area network.
  • LAN - A local area network.
  • MAN - A metropolitan area network.
  • WAN - A wide area network.

Network Models
  • Bus Network - Machines are all connected to a common communication line called a bus.
  • Star Network - One machine serves as a focal point to which all of the other machines are connected.

Protocols
  • CSMA/CD Ethernet - Stands for Carrier Sense, Mutiple Access with Collision Detection. It is the protocol that controls the right to transmit messages on a bus network and detects collisions.
  • CSMA/CA Wifi - Stands for Carrier Sense, Multiple Access with Collision Avoidance. It is the protocol that controls the right to transmit messages on a wireless star network and avoids collisions.

Combining Networks
  • Repeater - Device that connecs two buses and passes signals back and forth between the two without looking at the meaning of the signals.
  • Bridge - Device that connects two buses, but does not always pass signals. It looks at the destination address and only fowards the signal when it is for a computer on the other side.
  • Switch - Device that connects several buses rather than just two and works similar to a bridge.
  • Router - Handles the connection between networks to form an internet.
  • Gateway - The point where a network is linked to an internet.

Client/server Model
  • P2P: Peer to Peer - Services are provided both to and from multiple clients.
  • Distributed System - Consists of sotware units that execute processes on many computers.
  • Cloud Computing - Consists of many shared computers on a network that allocate use for clients on an as needed basis.

Internet Addressing
  • Tier 1 - Internet service providers that consist of very high-speed, high-capacity, and international wide area networks.
  • Tier 2 - Internet service providers that are more regional in scope and can have less capacity and speed.
  • Tier 3 - An access internet service provider that is an independent internet operated by a single entity that supplies internet access to individual homes or businesses.

IP Address
  • ICANN
  • DNS - Stands for domain name system and it is an internet-wide directory system.
  • TLD: Top-level domain - Suffix in the domain name for indentifying the domain's classification.
  • Country-code TLDs

Internet Protocols
  • FTP - Stands for file transfer protocol and is an application for accessing a computer from a great distance using a Telnet protocol or Secure Shell protocol.
  • HTTP - Stands for hyper text transfer protocol and defines how hypertext documents are transferred between browsers and webservers.
  • SMTP - Stands for simple mail transfer protocol and it defines how two computers on a network interact when transmitting an email message.

World Wide Web
  • URL - Stands for uniform resoure locator and each document is given a unique address so it can be located and retrieved on the internet.
  • Client-side Activities - Activities on the web that are performed by client.
  • Server-side Activities - Activities on the web that are performed by a server.

Internet Four-layer Software
  • Application Layer - Prepares messages, provides the destination address, and receives messages.
  • Transport Layer - Splits the messages into packets, collects packets, and then reassembles messages.
  • Network Layer - Assigns intermediate addresses to each of the packets, and detects if packets have reached their final destination.
  • Link Layer - Transfers and receives the packets.

Internet is:

  • Packet-switching Network - A network that groups and sends data in the form of small packets.
  • Fault Tolerant - Ability of a system to continue operating without interruption when one or more of its components fail.
  • Redundant - A system that has additional links to connect all nodes in case one of the links fail.

Cybersecurity
  • Malware - Software that is designed to disrupt, damage, or gain unauthorized access to a computer system.
  • Virus - Sotware that is designed to infect a computer by inserting itself into programs that are already on a computer.
  • Worm - A single program that tranfers through a network to a computer and then fowards copies of itself to other connected computers.
  • Trojan horse - A malicious program that disguises itself as a good program that will be downloaded by the user.
  • Spyware - Software that collects information about computer activities and reports back to the owner of the program.
  • Phising - Posing as a reputable organization in order to gain sensitive information for malicious purposes.
  • DDos attack - Stands for denial of service attack and is the process of overloading a computer system with messages in order to overwelm a targeted server.

Protection
  • Firewall - A program used for filtering messages passing through a point in a network.
  • Spam Filter - A firewall designed to block unwanted emails.
  • Proxy Server - Sotware that acts as an intermediary between a client and a server to sheild the client from adverse actions.
  • Antivirus Software - Used to detect and remove the known viruses and infections from a computer system.
  • Cryptography - The study of secure communications techniques that allow only the sender and intended recipient of a message to view its contents.
  • Public Key Encryption - Method of encrypting data with two different keys where one key is a public key and the other is private.
