Operating Systems Terminology
Term Definition
Operating System The software that controls the overall operation of the computer.
Job Queue An ordered list of jobs waiting to be processed by the computer system.
FIFO Stands for first in and first out order in a job queue. It is a method for organizing, handling, and manipulating elements in a computing system.
Interactive Processing Allows a user to enter instructions to the computer through a remote termimal while the computer is processing.
Time-sharing Allows multiple users to access a computer system at the same time.
Multiprogramming A method for implementing time sharing where time is divided into intervals and job execution is restricted to only one time interval at a time.
Application Software Programs that are used for performing tasks related the machine's utilization. The software performs a specific function directly for an end user or another application.
System Software Programs that perform tasks that are common to computer systems in general.
Utility Software Includes software programs that can extend or customize the capabilities of the software system.
Graphical User Interface Allows the manipulation of objects such as files and programs pictorially on a computer display screen using icons and input devices such as a mouse, touch pad, or stylus.
Shell A user interface that communicates with the user through textual messages using a keyboard and a monitor screen.
Kernel A computer program at the core of a computers operating system. It includes tasks such as process and memory management, file systems, device control, and networking.
Device Driver Software units that communicate with the controllers to carry out operations on any peripheral devices attached to the computer system.
File Manager Software component that coordinates the use of a computer's mass file storage system.
Memory Manager Software component that coordinates the use of a computer's main memory.
Virtual Memory The large fictional memory space created by paging, which is rotating programs and data back and forth between main memory and mass storage.
Boot Strapping Procedure used to start the operating system each time the computer is turned on. It transfers the operating system from mass storage into main memory.
ROM Stands for read only memory and are special nonvolatile memory cells that contain contents that are read only and cannot be altered.
Deadlock The condition where two or more computer processes are blocked from completing, since one is waiting for a resource that is allocated to another.
Spooling The process of holding data for output to a peripheral to a later and more convenient time.
Super User (or Admistrator) A user who has highly privileged access to an operating system. The user can alter settings with the operating system, modify software, adjust user privileges, and do other maintenance activities not allowed for regular users.
Multi-factor Authenication Requires more than one piece of information to gain access to a computer system, such as a password and a special access code.
